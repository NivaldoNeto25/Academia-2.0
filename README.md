# Sistema de Academia

Sistema desktop para gestão de academia, criado em Java com JavaFX.

# Funcionalidades
- Autenticação de usuário (ADM/Aluno)

- Cadastro e edição de alunos

- Cadastro, edição e exclusão de exercícios

- Cadastro e acompanhamento de planos de treino

- Registro de indicadores biomédicos (peso, altura, percentual de gordura, massa magra)

- Controle e persistência dos dados

# Tecnologias e Ferramentas
- Java 21

- JavaFX 21

- Maven

- JUnit 5 (Testes unitários)

- Mockito (Mocks para testes de business)

- Jacoco (Cobertura de testes)

- SonarCloud (Quality Gate)
